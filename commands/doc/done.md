# /doc done - 标记主题为已实现

标记当前或指定主题为「🚀 已实现」状态。

## 使用方法

```bash
/doc done [主题名]
```

## 参数

- `[主题名]` - 可选，若未指定则使用当前活跃主题

## 执行流程

1. 确认主题存在且状态为「✅ 已确认」
2. 更新需求文档状态为「🚀 已实现」
3. 立即更新 INDEX.md：状态列改为「🚀 已实现」，更新时间列
4. 告知用户："主题「XXX」已标记为已实现，索引已更新"

## 示例

```bash
# 标记当前主题为已实现
/doc done

# 标记指定主题为已实现
/doc done 用户认证功能
```
