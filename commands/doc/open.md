# /doc open [主题] - 切换主题

继续已有主题的讨论。

## 用法

```bash
/doc open [主题名]
```

## 参数

| 参数 | 说明 |
|------|------|
| `主题名` | 主题名称（支持模糊匹配，部分匹配即可）|

## 执行逻辑

1. 模糊匹配主题名
2. 读取对应需求文档的「背景与目标」「核心需求」部分
3. 输出摘要："已切换到主题「XXX」(状态：📝)。当前进展：..."

## 示例

```bash
/doc open 用户模块      # 匹配 "用户模块权限管理"
/doc open 支付          # 匹配 "支付系统重构"
```
